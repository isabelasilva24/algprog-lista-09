## Lista de Exercícios IX

Este repositório contém a resolução da Lista de Exercícios IX da disciplina de Algoritmos e Programação.

## Objetivo

O objetivo da atividade é identificar objetos do mundo real e descrever suas principais características, chamadas de atributos, e suas ações, chamadas de métodos.

## Exercício

Foram escolhidos 10 objetos do mundo real.  
Para cada objeto, foram definidos seus atributos e métodos.

## Objetos utilizados

- Pessoa
- Carro
- Animal
- Celular
- Computador
- Livro
- Bicicleta
- Televisão
- Geladeira
- Caneta

## Conceitos trabalhados

- Objetos
- Atributos
- Métodos
- Programação Orientada a Objetos
- Organização de informações
